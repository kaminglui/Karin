# Bridge runtime dependencies — pinned for reproducibility.

# IMPORTANT: torch is intentionally NOT pinned here. Install it first,
# from whichever source matches your hardware:
#   - Jetson / JetPack:  NVIDIA's supplied wheels (see setup.sh)
#   - x86 with CUDA:     pip install torch --index-url https://download.pytorch.org/whl/cu124
#   - CPU only:          pip install torch
# `pip install torch` via PyPI on ARM64/JetPack pulls a CPU wheel and
# silently breaks CUDA — the Dockerfile + setup.sh handle this.
